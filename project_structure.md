# Project Structure

```
blackmarket/
│
├── scrapers/         # Web scraping scripts for each marketplace
├── data/             # Raw and processed data storage
├── models/           # Anomaly detection and forecasting models
├── notebooks/        # Jupyter notebooks for EDA and visualization
├── utils/            # Utility/helper functions
├── requirements.txt  # Python dependencies
├── README.md         # Project overview
└── project_structure.md # This file
```

- **scrapers/**: Contains scripts for scraping different marketplaces.
- **data/**: Stores raw HTML, extracted CSVs, and cleaned datasets.
- **models/**: For saving trained models or scripts related to modeling.
- **notebooks/**: For exploratory data analysis and reporting.
- **utils/**: Helper functions (e.g., Tor connection, parsing, etc.).
